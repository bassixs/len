## Структура проекта (лен)

Визуально иерархия проекта выглядит так:

- `/` (корень)
  - `index.html`, `catalog.html`, `product.html`, ... — страницы сайта (главная, каталог, категории, товар, блог, контакты и т.д.)
  - `styles.css` — точка входа для стилей (подключает файлы из `css/`)
  - `vite.config.js` — конфигурация Vite (мультистраничный билд, Handlebars-партиалы)
  - `package.json` / `package-lock.json` — зависимости и скрипты (`dev`, `build`, `lint`, `format`, `check`)
  - `PROJECT_STRUCTURE.md` — это описание структуры
  - `ROADMAP.md` — планы по развитию проекта

- `/js`
  - `main.js` — главный вход фронтенда; на `DOMContentLoaded` инициализирует модули в зависимости от содержимого страницы
  - `/modules`
    - **Данные и утилиты**
      - `product-model.js`
        - общая модель товара: `normalizeProduct`, `formatPrice`, `resolveImageUrl`, `safeText`
        - используется в каталоге, карточке товара, при обогащении карточек
      - `cart-service.js`
        - работа с корзиной как с данными (без DOM): `getCart`, `saveCart`, `addItem`, `removeItem`, `updateItemQuantity`
        - расчёты: `cartSubtotal`, `getDeliveryCost`, `getCartTotals`
    - **UI-инфраструктура**
      - `ui-shell.js`
        - управление “слоями” UI (мобильное меню, фильтр, модалки, мини-корзина): `openLayer`, `closeLayer`, `closeAllLayers`
        - глобальная обработка `Escape` и блокировка скролла через класс `body.ui-locked`
      - `accordion.js`
        - универсальный аккордеон: `initAccordionGroup` для FAQ, описаний товара и других секций
    - **Основные UI-модули**
      - `navigation.js`
        - шапка и навигация: поведение при скролле, бургер-меню (через `ui-shell`), подсветка активной ссылки, плавный скролл
      - `catalog.js`
        - логика страницы каталога:
          - фильтры (группы, сайдбар, переключатель вида)
          - загрузка превью товаров из `public/data/products/index.json`
          - рендер карточек через `product-model`
      - `category-products.js`
        - логика страницы конкретной категории: загрузка товаров категории, фильтрация/пагинация (если есть), рендер сетки
      - `product.js`
        - страница товара:
          - загрузка товара по `id` из `data/products/*.json`
          - заполнение названия, цены, хлебных крошек
          - рендер галереи изображений и переключение превью
          - аккордеоны “Описание / Характеристики / Уход / Доставка” через `accordion.js`
          - управление выбором цвета/размера и количеством
      - `product-cards.js`
        - “обогащение” статичных карточек на страницах:
          - подстановка `data-product-id`, ссылок `product.html?id=...`, реальных картинок из `data/products`
          - работа с `index.json` и category JSON как с источником истины
      - `cart.js`
        - UI-слой корзины:
          - привязка `add-to-cart` кнопок к DOM и сбор данных товара (включая размер, цвет, ссылку)
          - мини-корзина (drawer) и тосты “товар добавлен” (через `ui-shell`)
          - рендер страницы корзины, изменение количества, удаление позиций
          - обновление иконки корзины в шапке
          - чекаут: подсчёт суммы, доставки, итога через `cart-service.getCartTotals`
      - `forms.js`
        - обработчики форм:
          - контактная форма (состояние кнопки “отправлено” через класс `btn--submitting`)
          - модалка “заказать звонок” (открытие/закрытие через `ui-shell`)
          - FAQ-аккордеон через `accordion.js`
      - `animations.js`
        - анимации появления (`IntersectionObserver`), наблюдение за динамически добавленными элементами
        - прелоад картинок, спец-эффекты для страницы “О нас” (картинка под курсором)
      - `sliders.js`
        - горизонтальный слайдер товаров (скролл, drag, кнопки навигации)

- `/public`
  - `/data/products`
    - `index.json` — список категорий, превью товаров, связь `productId → category`
    - `*.json` — товары по категориям (`home-textile`, `women`, `men`, `accessories`, `gifts`, `fabrics`, `socks`)
  - `/images`
    - картинки товаров, категорий, логотипы и прочие ассеты

- `/partials`
  - `header.html` — шапка, навигация, иконка корзины
  - `footer.html` — подвал сайта
  - `modals.html` — модалка звонка, мини-корзина, контейнер тостов
  - все подключаются в страницы через Handlebars (`{{> header }}`, `{{> footer }}`, `{{> modals }}`)

- `/css`
  - `base.css` — базовые стили, переменные, типографика
  - `buttons.css` — кнопки (`.btn`, модификаторы, состояния)
  - `header.css`, `hero.css`, `collections.css`, `products.css`, `about.css`, `cta.css`, `footer.css` — тематические блоки
  - `pages.css` — стили страниц (каталог, категория, продукт, блог, контакты, доставка, сервисы и т.п.)
  - `cart-drawer.css` — мини-корзина и тосты
  - `product-premium.css` — премиальная страница товара
  - `responsive.css` — адаптивная вёрстка
  - все они импортируются в `styles.css`
